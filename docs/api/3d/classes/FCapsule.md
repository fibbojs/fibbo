# Class: FCapsule

## Description

A simple capsule model in FibboJS.

## Example

```ts
import { FScene3d, FCapsule } from '@fibbojs/3d'

const scene = new FScene3d()

const capsule = new FCapsule(scene)
scene.addComponent(capsule)
```
