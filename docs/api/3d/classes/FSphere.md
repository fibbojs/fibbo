# Class: FSphere

## Description

A simple sphere model in Fibbo.

## Example

```ts
import { FScene, FSphere } from '@fibbojs/3d'

const scene = new FScene()

const sphere = new FSphere(scene)
scene.addComponent(sphere)
```
